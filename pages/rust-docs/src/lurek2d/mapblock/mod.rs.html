1//! Map-block procedural assembly system.
2//!
3//! - Builds tile maps from composable blocks using scripted placement.
4//! - Supports configurable tile slots (floor, roof, object, walls, custom).
5//! - Carcassonne-style neighbor edge matching for placement constraints.
6//! - Multi-level (Z-layers) for multi-storey maps.
7//! - TopDown and Isometric orientations (no hex).
8//! - Arbitrary map shapes (not limited to rectangles).
9//! - Output converts to standard `TileMap` for rendering.
10
