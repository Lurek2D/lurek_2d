1//! Generic graph/tree/DAG layout algorithms.
2//!
3//! Provides algorithms for positioning nodes in 2D space:
4//! - **Tree layout** — Reingold-Tilford algorithm for hierarchical trees
5//! - **DAG layout** — Sugiyama layered algorithm for directed acyclic graphs
6//! - **Force-directed** — Fruchterman-Reingold spring simulation for arbitrary graphs
7//! - **Grid alignment** — Post-processing snap-to-grid and centering
8//!
9//! Used by: pipeline visualization, dialog tree view, skill trees, node editors.
10
