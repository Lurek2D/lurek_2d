LTraitProfile

Lua handle for trait values with temporary modifiers and archetype lookup.

Docs home/lurek.ai
Class Example · Exact example · ai.lua

    local tp = lurek.ai.newTraitProfile()
    tp:set("courage", 0.7)

end
