Module csv 

Source
Expand description

CSV parsing and generation with configurable options. Parse CSV text or streams into SerialValue sequences of maps or arrays.

  • Serialize SerialValue back to CSV with configurable delimiter and header behavior.
  • Support both header-keyed (map rows) and index-only (sequence rows) modes.
