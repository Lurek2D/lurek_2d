Module ecs 

Source
Expand description

Exposes the ECS subsystem module. Lightweight ECS: entities with generational IDs, Lua-table components, tags, and blueprints.

  • Relationship graph for parent/child, ownership, and custom link types between entities.
  • Deep-copy and snapshot utilities for cloning Lua component tables.
