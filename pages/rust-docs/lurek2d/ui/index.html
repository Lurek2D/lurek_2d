Module ui 

Source
Expand description

Exposes the UI subsystem module. Immediate-mode GUI toolkit: containers, controls, extras, and theming.

  • Provides layout panels, interactive widgets, and data-bound context.
  • Optional TOML layout-loader feature behind a feature flag.
