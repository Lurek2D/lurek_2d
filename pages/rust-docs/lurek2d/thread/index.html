Exposes the threading subsystem module. Cross-thread messaging via typed MPMC channels for Lua VM isolation.

  • Fixed-size thread pool for CPU-bound background tasks.
  • Promise containers for single-value async results.
  • Worker harness owning secondary Lua VMs for parallel script execution.

Modules§

channel
Typed MPMC channel built on crossbeam-style semantics for cross-thread messages. Thread-safe MPMC channel for passing typed values between Lua VMs.
pool
Fixed-size thread pool for CPU-bound tasks dispatched from the game thread. Fixed-size worker pool backed by LuaThread instances sharing input/output channels.
promise
Single-value async result container shared between a producer thread and consumer. One-shot async computation that spawns a LuaThread and collects a single result.
worker
Worker-thread harness that owns a secondary Lua VM and processes message payloads. Worker VM lifecycle: spawn an OS thread with an isolated Lua VM, track Pending/Running/Completed/Error states.
