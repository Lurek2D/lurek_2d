Exposes the mod management subsystem module. Mod system entry point exposing lifecycle management for game mods.

  • Handles discovery, enabling/disabling, and Lua script integration of mods.
  • Game API registry for type-safe mod content declarations.
  • Sandboxing to restrict mod capabilities.
  • Instance loading from TOML content files.

Modules§

api_registry
Game API type registry and instance validation. Mod API registry: records which lurek.* namespaces are available to mod scripts.
api_schema
API schema types: fields, methods, asset requirements. Mod API schema: JSON-serialisable description of every lurek.* function signature.
mod_loader
TOML-based mod instance loading. Mod loader: discovers, validates, and loads mod packages from the mods directory.
mod_manager
Mod lifecycle management: discovery, enable/disable, and Lua integration. Mod registry: register, unregister, and look up mods by id or capability.
mod_sandbox
Sandbox environment restricting mod Lua API access and capabilities. Mod sandbox: restricts mod Lua API access to the declared capability set.
