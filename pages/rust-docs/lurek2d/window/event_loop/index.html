Module event_loop 

Source
Expand description

Event-loop helpers: display enumeration, monitor selection, window centering, and startup placement. Multi-monitor enumeration, display info snapshots, and primary-monitor detection.

  • Monitor selection with fallback logic (current → primary → first available).
  • Window centering and cross-display movement helpers.
  • Startup monitor resolution for initial window placement.
