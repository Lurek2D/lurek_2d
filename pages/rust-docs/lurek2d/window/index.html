Module window 

Source
Expand description

Exposes the window subsystem module. OS window lifecycle: creation, sizing, positioning, fullscreen, and DPI handling.

  • Multi-monitor support: display enumeration, selection, and window placement.
  • Virtual viewport: logical-to-pixel scaling and scale-mode selection.
