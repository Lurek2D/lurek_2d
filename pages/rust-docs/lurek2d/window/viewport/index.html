Module viewport

Virtual viewport scaling: logical-to-pixel mapping and scale-mode selection. Logical game viewport size queries and scale-mode staging.

  • Coordinate conversion between logical game space and physical screen pixels.
  • Viewport scale/offset snapshot via ScaleInfo.
