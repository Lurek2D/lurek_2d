Exposes the sprite subsystem module. Sprite, SpriteSheet, and SpriteBatch types for 2D rendering

  • Texture atlas parsing (TexturePacker JSON) and region lookup
  • Nine-slice panel geometry for scalable UI elements

atlas
Texture atlas region map and TexturePacker JSON parser. Named atlas region type (AtlasEntry) with pixel rect, rotation, and flip flags.
nine_slice
Nine-slice panel geometry for scalable UI borders and boxes. Nine-slice (9-patch) descriptor that splits a texture into corners, edges, and a center.
sprite
Core Sprite type with transform, tint, and region data. Single-sprite data type holding texture, position, scale, rotation, and colour tint.
sprite_batch
SpriteBatch: deferred draw-call accumulation for grouped sprite rendering. Deferred sprite draw-call collector bound to a single texture atlas.
sprite_sheet
SpriteSheet: uniform grid frame extraction from a single texture. Uniform grid frame extraction from a single texture via per-frame width/height.
