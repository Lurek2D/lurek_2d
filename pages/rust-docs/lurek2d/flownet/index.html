Module flownet

Expand description

Exposes the flownet subsystem module when the feature is enabled. Directed flownet container with typed nodes, edges, and item flow.

  • Supply/demand modeling, conversion rules, and overflow policies.
  • Pathfinding, simulation stepping, and event emission.
  • Render helpers for visual flownet output.
