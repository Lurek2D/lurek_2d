Expand description

Tile-based terrain map that synchronises static physics bodies with a World.

Fields§

§width: u32

Grid width in cells.

§height: u32

Grid height in cells.

§cell_size: f32

World units per cell side.

§offset_x: f32

World-space x origin of the grid.

§offset_y: f32

World-space y origin of the grid.
