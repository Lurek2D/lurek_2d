Exposes the terminal subsystem module. In-engine terminal emulator with ANSI escape-code support

  • Grid-based cell model, tab completion, and syntax highlighting
  • Converts terminal state into RenderCommand sequences for display
  • Widget primitives for composing custom terminal UIs

completion
Tab-completion provider interface and built-in implementations. Prefix-based tab-completion engine for the in-game terminal.
highlighter
Syntax-highlighting trait and Lua keyword highlighter. Pattern-based text highlighting: match literal strings and assign foreground/background colors.
render
Render helpers that convert terminal state to RenderCommand sequences. Render the composited terminal cell grid as a list of RenderCommand draw calls.
