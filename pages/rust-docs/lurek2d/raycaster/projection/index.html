Column projection math: wall slice height and screen coordinates. Wall-column projection from ray distance to screen-pixel height and vertical bounds.

  • Distance-based shading for depth fog attenuation.
  • Returns (wall_height, draw_start, draw_end) clamped to valid screen-pixel bounds.
