GPU texture upload helpers for province id, border index, and distance field maps. Province GPU upload helpers for id, border-index, and distance-field textures.

  • Consistent texture descriptors for R32Uint, R16Uint, and R8Unorm data.
  • Byte packing utilities used by upload paths and unit tests.
