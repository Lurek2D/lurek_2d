lurek2d::log

Module facade 

Source
Expand description

Structured log facade: global level, enabled checks, and dispatch to sink registry. Structured log dispatch with level, tag, and key-value fields.

  • Runtime log-level query and mutation via string names.
  • Level-filter check without emitting a message.

Functions§

enabled_for
Return true if the global log filter allows messages at level; returns false for unknown strings.
get_level
Return the current global log level as a string.
log_structured
Dispatch a structured log message at level with optional tag and fields; tag defaults to “Lua”.
set_level
Set the global log level from a string (“error”, “warn”, “info”, “debug”, “trace”).
