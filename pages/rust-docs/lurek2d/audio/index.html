Module audio 

Source
Expand description

Exposes the audio subsystem module. Audio subsystem module: mixer, buses, decoders, pools, and device enumeration.

  • Re-exports primary types: Mixer, Bus, Decoder, SoundData, SoundPool.
  • DSP effects are in crate::dsp; MIDI playback is in crate::midi.
