Module

lurek.devtools

`lurek.devtools` -- Developer tooling bindings for logs, profiling, frame stats, file watching, console state, watch expressions, snapshots, REPL, and debugger-style eval helpers. - Registers `lurek.devtools.*` functions and types via `register()`. - `LuaFileWatcher`: userdata type exposed to Lua. - `LuaReplConsole`: userdata type exposed to Lua. - Bridges 62 Lua-callable methods via `mlua`. - See `docs/specs/devtools.md` for the full API specification.
