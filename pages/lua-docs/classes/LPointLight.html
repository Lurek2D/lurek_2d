LPointLight

Lua-visible point light that illuminates nearby raycaster tiles and sprites with colored light and falloff.

Docs home/lurek.raycaster

set

LPointLight:set(x: number, y: number, r: number, g: number, b: number, radius: number, intensity: number)

Overwrites all properties of this point light in a single call.

methodsrc/lua_api/raycaster_api.rs:326
Example · Exact example · raycaster.lua
do
    local light = lurek.raycaster.newPointLight(2, 2, 1, 1, 1, 3, 1.0)
    light:set(8, 8, 0, 0, 1, 6, 2.0)
    local r, g, b = light:color()

    print("pos = " .. light:x() .. "," .. light:y())
    print("color = " .. r .. "," .. g .. "," .. b)
    print("radius = " .. light:radius() .. " intensity = " .. light:intensity())
end
