LBlendLayerSet

Lua-side blend layer set used to combine animation clips with weights and bone masks.

Docs home/lurek.animation
Class Example · Exact example · animation.lua
do
    local bls = lurek.animation.newBlendLayerSet()
    bls:addLayer("base", "idle", 1.0)
    print("layers added")
    print("layer count = " .. bls:len())
end
