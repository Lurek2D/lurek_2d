LPaletteLUT

Lua-side handle for palette color remapping.

Docs home/lurek.image
Class Example · Exact example · image.lua
do
    local lut = lurek.image.newPaletteLut()
    lut:setColor(255, 0, 0, 255, 0, 255, 0, 255)
    print("red → green mapping set")
end
