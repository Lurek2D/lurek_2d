Class

LIsoMap

Lua-side handle wrapping an `IsoMap` for isometric tile rendering with multi-level support and configurable part ordering.

Docs home/lurek.tilemap
Class Example · Exact example · tilemap.lua
do
    ---@type LIsoMap
    local iso = lurek.tilemap.newIsoMap(10, 10, 64, 32, 16, 4)
    local lvl = iso:addLevel()
    print("added level, count = " .. iso:getLevelCount())
end
