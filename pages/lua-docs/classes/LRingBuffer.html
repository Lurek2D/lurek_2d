Class

LRingBuffer

Lua-side fixed-capacity FIFO buffer that stores registry-protected Lua values.

Docs home/lurek.binary
Class Example · Exact example · binary.lua
do
    local rb = lurek.binary.newRingBuffer(3)
    rb:push("a")
    rb:push("b")
    local evicted = rb:push("d")
    print("evicted = " .. tostring(evicted))
    print("newest = " .. tostring(rb:peekNewest()))
end
