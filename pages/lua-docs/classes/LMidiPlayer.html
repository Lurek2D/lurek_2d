Class

LMidiPlayer

Lua-side wrapper around a MIDI file player with per-channel control and tempo scaling.

Docs home/lurek.audio
Class Example · Exact example · audio.lua
do
    local player = lurek.audio.newMidiPlayer()
    local path = "content/examples/assets/audio/sample_midi.mid"
    local ok = player:load(path)
    print("loaded = " .. tostring(ok))
end
