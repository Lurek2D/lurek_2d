Class

LHeightMap

Lua-visible height map that stores per-cell floor and ceiling offsets for variable-height raycaster levels.

Docs home/lurek.raycaster
Class Example · Exact example · raycaster.lua
do
    local hm = lurek.raycaster.newHeightMap(16, 16)
    hm:setFloor(3, 3, 0.2)

    print("floor = " .. hm:floorAt(3, 3))
end
