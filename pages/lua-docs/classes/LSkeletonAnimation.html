Class

LSkeletonAnimation

Lua-facing animation object containing bone timelines, keyframes, events, and easing curves.

Docs home/lurek.spine
Class Example · Exact example · spine.lua
do
    local anim = lurek.spine.newSkeletonAnimation("bob", 1.0)
    anim:addKeyframe(0, "y", 0.0, 0)
    anim:addKeyframe(0, "y", 1.0, 0, "ease_in")
    print("timeline count = " .. anim:getTimelineCount())
end
