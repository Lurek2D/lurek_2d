LAIDirector

Lua handle for an AI director that tracks encounter tension and pacing factors.

Docs home/lurek.ai
Class Example · Exact example · ai.lua
do
    local dir = lurek.ai.newAIDirector()
    dir:pushEvent(0.5)
    dir:pushEvent(0.8)
    print("events pushed, tension = " .. dir:tension())
end
