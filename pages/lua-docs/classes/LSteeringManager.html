Class

LSteeringManager

Lua handle for a steering behavior stack that combines movement forces for an agent.

Docs home/lurek.ai
Class Example · Exact example · ai.lua
do
  local steer = lurek.ai.newSteeringManager()
  steer:addSeek(400, 300, 1.0)
  local fx, fy = steer:calculate(50, 50, 0, 0, 100, 200, 1 / 60)
  print("LSteeringManager:addSeek: fx=" .. tostring(fx) .. " fy=" .. tostring(fy))
end

applyCustomSteering
