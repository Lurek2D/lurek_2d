LQuad

Rectangular sub-region of a texture, used for sprite sheets and atlas-based rendering.

Docs home/lurek.render
Class Example · Exact example · render.lua
do
    local quad = lurek.render.newQuad(0, 0, 32, 32, 128, 128)
    local x, y, w, h = quad:getViewport()
    print("quad viewport = " .. x .. "," .. y .. "," .. w .. "," .. h)
    print("quad type = " .. quad:type())
end
