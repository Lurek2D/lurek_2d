LDataWriter

Lua-side binary writer for sequential byte construction.

Docs home/lurek.binary
Class Example · Exact example · binary.lua
do
    local w = lurek.binary.newWriter()
    w:writeU8(255)
    print("after writeU8 len = " .. w:len())
    print("first byte = " .. string.byte(w:toBytes(), 1))
end
