Class

LQueue

Lua-facing FIFO queue with optional capacity limit. Supports enqueue/dequeue from both ends.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local q = lurek.patterns.newQueue(10)
    q:enqueue("msg1")
    q:enqueue("msg2")
    print("front = " .. tostring(q:front()))
    print("len = " .. q:len())
end
