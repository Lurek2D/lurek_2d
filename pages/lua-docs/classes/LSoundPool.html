Class

LSoundPool

Lua-side wrapper around a pre-allocated pool of identical sound voices for rapid fire effects.

Docs home/lurek.audio
Class Example · Exact example · audio.lua
do
    local path = "content/examples/assets/audio/sample_click.wav"
    local pool = lurek.audio.newPool(path, 4)
    local id = pool:play()
    print("playing voice id = " .. id)
end
