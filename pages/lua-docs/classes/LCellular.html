Class

LCellular

A cellular automaton simulation grid (sand, water, fire, gas, rock) for per-cell falling-sand style simulation.

Docs home/lurek.procgen
Class Example · Exact example · physics.lua
do
    local ca = lurek.physics.newCellular(32, 32)
    ca:setCell(5, 5, lurek.physics.CELL_SAND)
    print("cell", ca:getCell(5, 5))
    print("type", ca:type())
end
