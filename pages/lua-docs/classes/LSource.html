LSource

Lua-side wrapper around a loaded audio source (sound effect or music stream).

Docs home/lurek.audio
Class Example · Exact example · audio.lua
do
    local path = "content/examples/assets/audio/sample_click.wav"
    local src = lurek.audio.newSource(path, "static")
    src:play()
    print("playing via method = " .. tostring(src:isPlaying()))
end
