Class

LSimpleState

Lua-facing finite state machine with enter/exit/update callbacks per state.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local fsm = lurek.patterns.newSimpleState()
    fsm:addState("idle", {
        enter = function()
            print("enter idle")
        end
    })
    fsm:addState("walk", {
        enter = function()
            print("enter walk")
        end
    })
    print("has walk = " .. tostring(fsm:hasState("walk")))
    print("states = " .. #fsm:getStates())
end

update

LSimpleState:update(dt: number)

Call the current state's update callback with the frame delta time.

methodsrc/lua_api/patterns_api.rs:667
Example · Exact example · patterns.lua
do
    local fsm = lurek.patterns.newSimpleState()
    fsm:addState("idle", {
        update = function(dt)
