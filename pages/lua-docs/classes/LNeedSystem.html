LNeedSystem

Lua handle for decaying needs and urgency selection.

Docs home/lurek.ai

do

    ns:addNeed("hunger", 0.1, 0.7, 2.0)
    ns:addNeed("thirst", 0.15, 0.6, 1.5)
    print("needs registered")
end
