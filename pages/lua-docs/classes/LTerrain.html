Class

LTerrain

A destructible terrain map backed by a grid of solid/empty cells. Generates physics colliders on flush.

Docs home/lurek.physics
Class Example · Exact example · physics.lua
do
    local world = lurek.physics.newWorld(0, 400)
    local terrain = lurek.physics.newTerrain(64, 64, 8, world)
    terrain:setCell(5, 5, true)
    print("cell", terrain:getCell(5, 5))
    print("dirty", terrain:isDirty())
end
