Class

LUiWidget

Creates a Lua table representing a widget with all shared base methods common to every widget type.

Docs home/lurek.ui
Class Example · Exact example · ui.lua
do
    local w = lurek.ui.newCustomWidget({width=100, height=50})
    w:slideIn(0, -50)
    w:slideOut(0, 50)
    local t = w:type()
    local ok = w:typeOf("LUiWidget")
    w:unbind()
    print("slideIn/slideOut ok; type:", t, "typeOf:", ok)
end
