LMap

Lua-facing string-keyed dictionary (map) with keys(), values(), entries(), and merge operations.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local map = lurek.patterns.newMap()
    map:set("name", "hero")
    map:set("level", 5)
    print("name = " .. tostring(map:get("name")))
    print("len = " .. map:len())
end

merge

LMap:merge(other: LMap)

Copy all entries from another LMap into this map. Existing keys are overwritten.

methodsrc/lua_api/patterns_api.rs:2468
Example · Exact example · patterns.lua
do
