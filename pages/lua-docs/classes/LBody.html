Class

LBody

A handle to a single physics body in the world, providing per-body manipulation methods.

Docs home/lurek.physics
Class Example · Exact example · physics.lua
do
    local world = lurek.physics.newWorld(0, 9.8)
    local body = lurek.physics.newBody(world, 100, 100, "dynamic")
    print("id", body:getId())
    print("position", body:getPosition())
end
