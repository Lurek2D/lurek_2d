Class

LParallaxLayer

Lua-side wrapper for a parallax layer and shared render state.

Docs home/lurek.parallax
Class Example · Exact example · parallax.lua
do
    local img = lurek.render.newImage("content/examples/assets/images/sample_texture.png")
    local layer = lurek.parallax.newLayer({
        texture = img,
        scroll_factor_x = 0.5,
        scroll_factor_y = 0.2,
        z = -1,
    })
    print(layer:type())
end
