LStack

Lua-facing LIFO stack with optional capacity limit. Supports push/pop from both ends.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local st = lurek.patterns.newStack(100)
    st:push("first")
    st:push("second")
    print("peek = " .. tostring(st:peek()))
    print("len = " .. st:len())
end
