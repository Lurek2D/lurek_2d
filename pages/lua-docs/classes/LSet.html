LSet

Lua-facing string set with add/remove/has operations and set algebra (union, intersection).

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local set = lurek.patterns.newSet()
    set:add("fire")
    set:add("ice")
    print("len = " .. set:len())
    print("has fire = " .. tostring(set:has("fire")))
end
