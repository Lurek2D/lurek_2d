Class

LBlackboard

Lua-facing shared key-value blackboard supporting bool/number/string values with watchers for reactive game logic.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local bb = lurek.patterns.newBlackboard("game_state")
    bb:set("score", 10)
    bb:set("score", 20)
    print("score = " .. tostring(bb:get("score")))
    print("revision = " .. bb:getRevision())
end

unwatch

LBlackboard:unwatch(id: integer)

Remove a previously registered watcher by its ID.

methodsrc/lua_api/patterns_api.rs:846
Example · Exact example · patterns.lua
do
    local bb = lurek.patterns.newBlackboard("game_state")
    local watch_id = bb:watch("score", function(key, value)
        print(key .. " = " .. tostring(value))
    end)
    bb:set("score", 10)
    bb:unwatch(watch_id)
    bb:set("score", 20)
    print("revision = " .. bb:getRevision())
end

watch

LBlackboard:watch(key: string, callback: function) -> number

Register a watcher callback that fires whenever the specified key changes. Use `"*"` to watch all keys.

methodsrc/lua_api/patterns_api.rs:828
Example · Exact example · patterns.lua
do
    local bb = lurek.patterns.newBlackboard("game_state")
    local watch_id = bb:watch("score", function(key, value)
        print(key .. " = " .. tostring(value))
    end)
    bb:set("score", 10)
    bb:set("score", 25)
    print("revision = " .. bb:getRevision())
    bb:unwatch(watch_id)
end
