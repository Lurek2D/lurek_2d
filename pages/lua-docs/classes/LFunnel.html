Class

LFunnel

Lua-facing batching funnel that collects events over a time window and flushes them together.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local funnel = lurek.patterns.newFunnel(1.0, 5, "damage_log")
    funnel:onFlush(function(entries)
        print("flushed = " .. #entries)
    end)
    funnel:push("fire", 10)
    funnel:push("ice", 5)
    funnel:update(1.1)
    print("pending = " .. funnel:pendingCount())
    print("flush count = " .. funnel:getFlushCount())
end
