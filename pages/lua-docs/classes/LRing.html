Class

LRing

Lua-facing fixed-size ring buffer for numeric or string values. Oldest entries are overwritten when full.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local ring = lurek.patterns.newRing(5, "fps_samples")
    ring:push(60)
    ring:push(58)
    ring:push(62)
    print("len = " .. ring:len())
    print("sum = " .. ring:sum())
end
