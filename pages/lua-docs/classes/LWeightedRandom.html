Class

LWeightedRandom

Lua-facing weighted random selection pool. Add items with weights and pick random selections.

Docs home/lurek.patterns
Class Example · Exact example · patterns.lua
do
    local wr = lurek.patterns.newWeightedRandom()
    wr:add(10, "common", "common_loot")
    wr:add(3, "rare", "rare_loot")
    print("items = " .. wr:len())
    print("total = " .. wr:totalWeight())
end
