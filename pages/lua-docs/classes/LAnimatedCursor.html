LAnimatedCursor

Lua userdata representing an animated cursor that cycles through image frames.

Docs home/lurek.cursor
Class Example · Exact example · cursor.lua
do
    local c = lurek.cursor.newAnimated(true)
    local frame = lurek.cursor.newCustom(16, 16, 0, 0)
    c:addFrame(frame, 100)
    print("LAnimatedCursor:addFrame count=" .. c:frameCount())
end
