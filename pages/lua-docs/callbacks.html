Global Index

Callbacks

Global lifecycle, input, and platform callbacks. These are not module methods; they are special runtime entry points the engine looks up directly on the global lurek table.

Runtime Hooks

Use these as stable entry points for frame flow, input, focus, and shutdown work.

function lurek.process_physics(dt)

Example · Module example · generated callback skeleton
