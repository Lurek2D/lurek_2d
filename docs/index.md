## Quick start

```bash
git clone https://github.com/Lurek2D/lurek_2d
cd lurek_2d
cargo run -- content/games/platformer
```

## Stack

| Layer | Library |
|---|---|
| Language | Rust 1.78+ |
| Scripting | LuaJIT via mlua 0.9 |
| Graphics | wgpu 22 |
| Physics | rapier2d 0.32 |
| Audio | rodio 0.17 |
| Font | fontdue 0.9 |
| Platform | winit 0.30 |
